# Käyttöohje

Tämä on sovellus lintuhavaintojen tallentamiseen.

## Aloitus

Käyttö aloitetaan luomalla käyttäjätunnus, jolla palveluun kirjaudutaan.


## Havaintojen tallennus

Tallentaminen tapahtuu lomakkeella, johon käyttäjä syöttää tiedot havainnosta, esimerkiksi lajin, paikan, ajan, määrän ja muita lisätietoja.
